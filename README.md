# brewino-privacy

Privacy policy for **Brewino: Gaggiuino Companion**, an unofficial iOS companion app for espresso machines running the open-source Gaggiuino firmware.

Published by [BrewByte](https://github.com/brew-byte).

## View the Policy
View the hosted version: [brew-byte.github.io/brewino-privacy](https://brew-byte.github.io/brewino-privacy)

## Summary

Brewino does not collect, store, or transmit any personal data. The app communicates only with your Gaggiuino espresso machine over your local Wi-Fi network. No analytics, no tracking, no external servers.

## Questions?

If you have questions or concerns about the privacy policy, please [open an issue](https://github.com/brew-byte/brewino-privacy/issues).

## Affiliation

Brewino is an independent, third-party app and is not affiliated with, endorsed by, or sponsored by the Gaggiuino open-source project or Gaggia S.p.A.
